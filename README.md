# Llanquihue Tour

## Descripción

Sistema desarrollado en Java para gestionar clientes, guías, proveedores, tours y reservas de una agencia de turismo.

El proyecto fue desarrollado utilizando Programación Orientada a Objetos.

## Principios utilizados

- Encapsulamiento
- Herencia
- Polimorfismo
- Interfaces
- Composición
- Excepciones
- Colecciones (ArrayList)
- Lectura de archivos

## Estructura

app
model
data
utils
interfaces
excepciones
archivos

## Cómo ejecutar el progrma

1. Clonar el repositorio.
2. Abrir el proyecto en IntelliJ IDEA.
3. Ejecutar Main.java.
